# Lab-8-ML

This lab applies the K-Nearest Neighbors (KNN) algorithm on a dataset to classify data into two classes. First, the data was explored using a pairplot to understand the relationships between features. Then, the features were standardized using StandardScaler to ensure all variables are on the same scale, which is important for KNN.

After that, the dataset was split into training and testing sets. A KNN model was trained with different values of K, and the model performance was evaluated using confusion matrix and classification report. The elbow method was used to find the best K value by plotting the error rate against different K values.

Finally, the model was retrained using the optimal K value, which improved the accuracy and overall performance. The results showed that choosing a suitable K value significantly affects the model performance.
